## 项目简介

全员差评BooHub是一款基于Nervos的dAPP。BooHub的建立受到一些新闻事件的启发，致力于搭建一套造假成本高、差评无负担的商户评价系统。

## 评价机制 ##

1. 付费评价

2. 验证交易

3. 差评辩解
##  验证体系：##
1. 发票（不支持图片）

2. 支付宝交易流水（限制频率bulabula）

## 商家辩解： ##
1. 有时限

2. 有差评时间溯源
